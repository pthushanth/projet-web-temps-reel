# projet-web-temps-reel


## Run command


Start the server
```
make start
```
or
```
docker-compose up -d 
cd nodejs_server 
npx sequelize db:migrate 
npx sequelize db:seed:all 
```
Restart server
```
make restart
```
stop server
```
make stop
```
